Detailed Description

RF_PACKAGE union.

RF_PACKAGE Format for communication between the transmitter and receiver.
the Package_Header can be customized for different packages.
the default Package_Header value used for controller data is 0x0000001D.
A package_Header value consists of 4 bytes:

  • byte1: MessageType.
  • byte2: receiverID.
  • byte3: TransferID.
  • byte4: Amount of data transfered. byte2 and byte3 are unused in the default header. byte1 is 0x00 for the default header for speed. The following 7 ints, or 24 bytes of data can be used freely.
