BimTi Thrashbots Controller & RF library
Used for TrashBots as a general library for the Controller PCB & RF communication.
RF_Handler/RF24Handler_Receiver/RF24Handler_Receiver.ino

This is an example of how to use the RFHandler class as a receiver. it defines the pins the nRF24l01 is connected to, in this case an Arduino 2560 is used Then it passes these values and the name of the device to the RFHandler constructor. In the setup the RFHandler::begin() is called and the name of the transmitter is passed to open a link to the transmitter. In the loop it continually checks if a packet was received, and if that is the case it prints the value. This example is meant to be used inconjunction with the example of RF_Handler/Controller_Transmitter/Controller_Transmitter.ino

#include <BimTi_RFHandler.h>
// ATMega2560 CE & CSN pins
#define CE 48
#define CSN 53
// Setup the RFHandler. The device name in this case is BattleBotTeam__Tx
RFHandler RH(CE, CSN, "BattleBotTeam__Rx");
void setup() {
Serial.begin(115200);
// initialize the nRF24l01 module and open a link to BattleBotTeam__Tx
if(!RH.begin("BattleBotTeam__Tx")){
while(1);
}
Serial.println("starting listening");
}
void loop() {
RF_PACKAGE buff;
// check if a packet was received
if(RH.getPacket(&buff)){
// read and display packet in serial monitor
Serial.println(buff.Package_Header, HEX);
Serial.println(String(buff.Pot[0]) + " | " + String(buff.Pot[1]));
Serial.println(String(buff.Joystick[0][0]) + " " + String(buff.Joystick[0][1]) + " | " + String(buff.Joystick[1][0]) + " " + String(buff.Joystick[1][1]));
Serial.println(buff.toggles, BIN);
}
}
