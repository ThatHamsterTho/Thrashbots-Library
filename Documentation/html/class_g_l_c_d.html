◆ clear()

void GLCD::clear ( )

clears the display.

clears the display by setting all bytes to 0x00 in display ram.

Remarks
note: if the clear function is called frequently the screen may start. to flikker, to prevent this use clearLine to clear the line you want to write to. You can also get the current cursor position through the getCursor function. If you want to clear the current line of the cursor use clearLine();
